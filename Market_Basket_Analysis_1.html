Market Basket Analysis - Parte 1¶



Esta a parte 1 (anlise exploratria) de um dos projetos de concluso do mdulo Business Analytics da Formao Cientista de Dados da Data Science Academy (DSA)

Definição do problema¶

Nesse projeto, utilizarei os dados annimos fornecidos pelo instacart para prever quais produtos adquiridos anteriormente estaro no prximo pedido de um usurio. O instacart um aplicativo de compras online, onde os compradores selecionam os produtos no app e a entrega feita na loja mais prxima voc.

Nessa primeira fase farei a anlise exploratria dos dados para entender o contedo dos datasets e o perfil de compra dos usurios.

The Instacart Online Grocery Shopping Dataset 2017, Accessed from:

https://www.instacart.com/datasets/grocery-shopping-2017 on 25/05/2020.

Post do VP da Instacart definindo o problema em detalhes:

https://tech.instacart.com/3-million-instacart-orders-open-sourced-d40d29ead6f2?gi=41f7b19cd164

# Funo - Proporo dos valores missing
propmiss = function(dataframe) {
  m = sapply(dataframe, function(x) {
    data.frame(
      nmiss = sum(is.na(x)),  
      n=length(x),
      propmiss = sum(is.na(x))/length(x) 
    )
  })
  d = data.frame(t(m))
  d = sapply(d, unlist)
  d = as.data.frame(d)
  d$variable = row.names(d)
  row.names(d) = NULL
  d = cbind(d[ncol(d)],d[-ncol(d)])
  return(d[order(d$propmiss), ])
}
